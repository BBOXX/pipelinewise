Taps (Data Sources)¶

PipelineWise can replicate data from the following data sources:

  • MySQL (Tap MySQL)

  • MariaDB (Tap MySQL)

  • PostgreSQL (Tap PostgreSQL)

  • Zendesk (Tap Zendesk)

  • Kafka stream (Tap Kafka)

  • CSV file on S3 (Target S3 CSV)
